## Prerequisites

- Node = v14
- pnpm

## Develop

Change `.env.example` to `.env`

Ask [me](https://twitter.com/agalliosamai) for the env.

```bash
# install
$ pnpm i

# development build
$ pnpm dev

# production build
$ pnpm build
```
