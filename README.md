# ina-covid-bed

[![website preview](/public/readme.png)](https://bed.ina-covid.com)

## Prerequisites

- Node > v14
- pnpm

## Develop

Change `.env.example` to `.env`

```
NEXT_PUBLIC_MAPBOX=
NEXT_REDIS_URL=
```

Ask [@agalliosamai](https://twitter.com/agalliosamai) for REDIS URL

```bash
# install
$ pnpm i

# development build
$ pnpm dev

# production build
$ pnpm build
```
